## Online store of computer games, created using Django.

### Implemented systems:
+ Login with Steam
+ Login with GitHub
+ Login with Google
+ Shopping cart system
+ Comment and feedback system
+ Integration of reCAPTCHA
+ Ability to add, edit and delete posts
